About

The Goal of this project is to provide a clean, clear way to find a movie to watch, void of ads and a bunch of unnecessary links.

With that in mind it had to use box art images, bits of movie info, and a responsive layout.

To achieve this goal, I used a minimalistic css framework called Milligram which I modified. I built my own modal, pagination, and navbar responsive menu.

Because of this approach of writing things from scratch, I encountered things I didn't expect like issues with touch on iOS.

On a positive note this project also gives me chance to play with Angular 2 and Observables.
